How to host on GitHub Pages (free, permanent link)

The dashboard is a single HTML file. Host it on GitHub Pages and anyone with your link can open it in their browser instantly — no login needed, free forever.

1
Create a free GitHub account
Go to github.com/signup and sign up. Free plan is all you need — no credit card.
2
Create a new repository
Click the + icon top-right → New repository → name it gml-tracker → set to Public → tick Add a README file → Create repository.
3
Upload the dashboard file
Inside your new repo, click Add file → Upload files → drag in the downloaded gml-command-centre.html → rename it to index.html → click Commit changes.
4
Enable GitHub Pages
Go to your repo → Settings → Pages (left sidebar) → under Source select Deploy from a branch → Branch: main → folder: / (root) → click Save. Wait ~60 seconds.
5
Get your live link
Go back to Settings → Pages — your URL will appear: https://YOUR-USERNAME.github.io/gml-tracker. That's your permanent link — share it with anyone!
6
Keeping it updated
When Alex sends new tasks: click ↻ Sync Slack → click Download latest dashboard → go to your GitHub repo → click index.html → pencil icon → drag new file → Commit changes. Link stays the same!

API key — for live Slack sync

The Sync Slack button uses an Anthropic API key to parse Alex's messages with AI. Your key is stored only in your browser's local storage — it is never sent anywhere except directly to Anthropic's API.

Get your key at console.anthropic.com → API Keys → Create key
