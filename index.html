📤 How to share this dashboard

⚡ Option 1 — Netlify Drop (easiest, free, instant)

No account needed. Anyone with the link can view the full dashboard in their browser.

  1. Click "⬇ Export & Share" in the top bar — downloads the HTML file
  2. Go to app.netlify.com/drop in your browser
  3. Drag and drop the HTML file onto the page
  4. Netlify gives you a link — copy and share it with anyone

📁 Option 2 — GitHub Pages (free, permanent link)

Best for a stable link that doesn't change.

  1. Create a free account at github.com
  2. Create a new repository, upload the HTML file as index.html
  3. Go to Settings → Pages → set source to main branch
  4. Your link: yourusername.github.io/repo-name

📧 Option 3 — Send the file directly

The HTML file works 100% offline. Email or WhatsApp it — the recipient just opens it in any browser. All tasks and data are embedded inside the file. To share updates, re-export and re-share.

💡 Tip — keeping everyone in sync

Each person who opens the file sees the same task list, but their check-offs are saved only in their own browser. To push updates (new tasks from Alex), re-export and re-share the updated file. This takes about 10 seconds.
